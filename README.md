# AI Art Generator Platform

Платформа для создания изображений с помощью искусственного интеллекта и их публикации.

## Функциональность
- Генерация изображений через Stability AI API
- Галерея с работами пользователей
- Административная панель
- Система авторизации
- Адаптивный дизайн

## Установка
1. Клонировать репозиторий
2. Установить зависимости: `npm install`
3. Создать файл `.env` с API ключом
4. Запустить: `npm start`
